## Runner and Customer Experience

**1. How many runners signed up for each 1 week period? (i.e. week starts 2021-01-01)**

SQL Query:
````sql
    SELECT 
    	EXTRACT(WEEK FROM r.registration_date) AS signup_week,
    	COUNT(r.runner_id) as num_runners
    FROM runners as r
    GROUP BY signup_week
````
Result:
| signup_week | num_runners |
| ----------- | ----------- |
| 53          | 2           |
| 1           | 1           |
| 2           | 1           |
- 2 runners signed up on week 53, 1 runner signed up on weeks 1 & 2.

**2. What was the average time in minutes it took for each runner to arrive at the Pizza Runner HQ to pickup the order?**

SQL Query:
````sql
    WITH pickup_speed AS (
      SELECT 
      	ro.order_id,
      	ro.runner_id,
      	co.order_time,
       	ro.pickup_time,
      	EXTRACT(EPOCH FROM (ro.pickup_time::timestamp - co.order_time::timestamp))/60 AS minutes_to_pickup
      FROM runner_orders as ro
      LEFT JOIN customer_orders as co
      ON ro.order_id = co.order_id
      WHERE ro.cancellation IS NULL
     )  
        
    SELECT 
    	ps.runner_id,
        ROUND(AVG(ps.minutes_to_pickup)::numeric, 2) AS runner_avg
    FROM pickup_speed as ps
    GROUP BY ps.runner_id
    ORDER BY ps.runner_id ASC
````
Result:
| runner_id | runner_avg |
| --------- | ---------- |
| 1         | 15.68      |
| 2         | 23.72      |
| 3         | 10.47      |
- Runner 1 averaged 15.68 minutes to pick up a pizza for delivery, Runner 2 averaged 23.72 minutes, and Runner 3 averaged 10.47 minutes.
