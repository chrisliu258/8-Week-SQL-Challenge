## Runner and Customer Experience

**1. How many runners signed up for each 1 week period? (i.e. week starts 2021-01-01)**

SQL Query:
````sql
    SELECT 
    	EXTRACT(WEEK FROM r.registration_date) AS signup_week,
    	COUNT(r.runner_id) as num_runners
    FROM runners as r
    GROUP BY signup_week
````
Result:
| signup_week | num_runners |
| ----------- | ----------- |
| 53          | 2           |
| 1           | 1           |
| 2           | 1           |
- 2 runners signed up on week 53, 1 runner signed up on weeks 1 & 2.

**2. What was the average time in minutes it took for each runner to arrive at the Pizza Runner HQ to pickup the order?**

SQL Query:
````sql
    WITH pickup_speed AS (
      SELECT 
      	ro.order_id,
      	ro.runner_id,
      	co.order_time,
       	ro.pickup_time,
      	EXTRACT(EPOCH FROM (ro.pickup_time::timestamp - co.order_time::timestamp))/60 AS minutes_to_pickup
      FROM runner_orders as ro
      LEFT JOIN customer_orders as co
      ON ro.order_id = co.order_id
      WHERE ro.cancellation IS NULL
     )  
        
    SELECT 
    	ps.runner_id,
        ROUND(AVG(ps.minutes_to_pickup)::numeric, 2) AS runner_avg
    FROM pickup_speed as ps
    GROUP BY ps.runner_id
    ORDER BY ps.runner_id ASC
````
Result:
| runner_id | runner_avg |
| --------- | ---------- |
| 1         | 15.68      |
| 2         | 23.72      |
| 3         | 10.47      |
- Runner 1 averaged 15.68 minutes to pick up a pizza for delivery, Runner 2 averaged 23.72 minutes, and Runner 3 averaged 10.47 minutes.

**3. Is there any relationship between the number of pizzas and how long the order takes to prepare?**

SQL Query:
````sql
    WITH cook_time AS (
      SELECT 
      	ro.order_id,
      	COUNT(co.order_id) AS order_size,
      	co.order_time,
       	ro.pickup_time,
      	EXTRACT(EPOCH FROM (ro.pickup_time::timestamp - co.order_time::timestamp))/60 AS minutes_to_pickup
      FROM runner_orders as ro
      LEFT JOIN customer_orders as co
      ON ro.order_id = co.order_id
      WHERE ro.cancellation IS NULL
      GROUP BY ro.order_id, co.order_time, ro.pickup_time
      ORDER BY ro.order_id ASC
     )
    
           
    SELECT 
    	order_size,
        ROUND(AVG(minutes_to_pickup)::numeric, 2) as avg_prep_time
    FROM cook_time
    GROUP BY order_size
    ORDER BY order_size ASC
````
Result:
| order_size | avg_prep_time |
| ---------- | ------------- |
| 1          | 12.36         |
| 2          | 18.38         |
| 3          | 29.28         |

- There appears to be a positive correlation between order size and time. 1 pizza averaged 12.36 minutes to pickup, 2 pizzas 18.28 minutes, and 3 pizzas went up to 29.29 minutes.

**4. What was the average distance travelled for each customer?**

SQL Query:
````sql
    SELECT 
    	co.customer_id,
        ROUND(AVG(ro.distance::numeric), 2) AS avg_distance
    FROM customer_orders as co
    LEFT JOIN runner_orders as ro
    ON co.order_id = ro.order_id
    WHERE ro.cancellation IS NULL
    GROUP BY customer_id
````
Result:
| customer_id | avg_distance |
| ----------- | ------------ |
| 101         | 20.00        |
| 102         | 16.73        |
| 103         | 23.40        |
| 104         | 10.00        |
| 105         | 25.00        |
- Customer 101 averaged 20km distance from the pizza place, 102 averaged 16.73km, 103 averaged 23.40km, 104 was closest at 10km, and 105 furthest at 25km.

**5. What was the difference between the longest and shortest delivery times for all orders?**

SQL Query:
````sql
    SELECT 
    	MAX(duration::numeric) - MIN(duration::numeric) AS duration_diff
    FROM runner_orders
    WHERE duration IS NOT NULL
````
Result:
| duration_diff |
| ------------- |
| 30            |
- The difference between the longest and shortest delivery was 30 minutes.

**6. What was the average speed for each runner for each delivery and do you notice any trend for these values?**

SQL Query:
````sql
    SELECT 
    	co.order_id, 
        co.customer_id,
        ro.runner_id,
        ro.distance, 
        ro.duration,
        COUNT(co.order_id) AS order_size,
        ROUND((ro.distance::numeric/ro.duration::numeric * 60), 2) AS km_per_hour
    FROM customer_orders as co
    LEFT JOIN runner_orders as ro
    ON co.order_id = ro.order_id
    WHERE ro.cancellation IS NULL
    GROUP BY co.order_id, co.customer_id, ro.runner_id, ro.distance, ro.duration
    ORDER BY ro.runner_id ASC, co.order_id ASC
````

Result:
| order_id | customer_id | runner_id | distance | duration | order_size | km_per_hour |
| -------- | ----------- | --------- | -------- | -------- | ---------- | ----------- |
| 1        | 101         | 1         | 20       | 32       | 1          | 37.50       |
| 2        | 101         | 1         | 20       | 27       | 1          | 44.44       |
| 3        | 102         | 1         | 13.4     | 20       | 2          | 40.20       |
| 10       | 104         | 1         | 10       | 10       | 2          | 60.00       |
| 4        | 103         | 2         | 23.4     | 40       | 3          | 35.10       |
| 7        | 105         | 2         | 25       | 25       | 1          | 60.00       |
| 8        | 102         | 2         | 23.4     | 15       | 1          | 93.60       |
| 5        | 104         | 3         | 10       | 15       | 1          | 40.00       |

- Runner 1 averaged between 37.5 and 60 km/h, runner 2 between 35.1 and 93.6 km/h, and runner 3 averaged 40 km/h for their one delivery. There doesn't appear to be any strong correlation between their average speed and distance, duration, or order size.

**7. What is the successful delivery percentage for each runner?**

SQL Query:
````sql
    SELECT 
      runner_id, 
      ROUND(100 * SUM(
        CASE 
        	WHEN cancellation IS NOT NULL THEN 0
        	ELSE 1 
        END) / COUNT(runner_orders), 2) AS delivery_percentage
    FROM runner_orders
    GROUP BY runner_id
    ORDER BY runner_id
````
Result:
| runner_id | delivery_percentage |
| --------- | ------------------- |
| 1         | 100.00              |
| 2         | 75.00               |
| 3         | 50.00               |

- Runner 1 delivered all orders, runner 2 delivered 75% of orders, runner 3  delivered 50% of orders.
